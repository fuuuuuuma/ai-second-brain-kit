# AGENTS.md

> Codex / Gemini / その他のエージェント用の起点ファイルです。
> この Vault のルールは [`CLAUDE.md`](CLAUDE.md) に集約されています。**まず CLAUDE.md を読んでから作業を始めてください。**

## 最低限ここだけは守る

1. セッション開始時に `CLAUDE.md` → `rules/corrections.md` → `rules/mistakes.md` → `wiki/index.md` を読む。
2. オーナーに訂正されたら `rules/corrections.md` に3行で追記する（次回以降も守る）。
3. `raw/` は読むだけ（削除・改名しない）。構造化した知識は `wiki/` に書く。
4. 答えは `reports/` にファイルで残し、`wiki/` ページにリンクする。
5. 破壊的操作（削除・上書き・大量改名）の前は確認する。
6. 日付・数値・固有名詞は確証がなければ「未確認」と明示する。

詳細はすべて [`CLAUDE.md`](CLAUDE.md) に従うこと。
