# mistakes.md — やらかしと再発防止

> AIが実際にやってしまった失敗を記録し、二度と繰り返さないためのルールに変える場所です。
> `corrections.md` が「先回りの指示」なら、こちらは「起きてしまった失敗の振り返り」です。

## 書き方（フォーマット）

```
## YYYY-MM-DD
- やらかし：（何をして、何が問題だったか）
- 原因：（なぜそうなったか。一言で）
- 再発防止：（次から何を確認/実行するか。具体的なトリガー付きで）
```

## 運用ルール（AI向け）

- 同じミスを2回指摘されたら、必ずここに記録する（1回でも重大なら記録）。
- 「気をつける」で終わらせない。**いつ・何を確認するか**まで書く。
- 解決して定着したルールは `corrections.md` か `CLAUDE.md` に昇格させ、ここからは整理してよい。
